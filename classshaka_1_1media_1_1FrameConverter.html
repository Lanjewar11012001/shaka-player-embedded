bool shaka::media::FrameConverter::ConvertFrame ( const AVFrame *  frame,

Converts a frame to the given pixel format, using swscale. The returned data is only valid until this method is called again or when this object is destroyed.

Parameters
frameThe frame to be converted.
dataA pointer that will be set to point to the new data of the converted frame if conversion is successful.
linesizeA pointer that will be set to point to the new linesizes of the converted frame if conversion is successful.
desired_pixel_formatThe pixel format to convert to.
Returns
True if the conversion was successful, false otherwise.
